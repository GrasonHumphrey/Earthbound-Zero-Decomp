PSI data
Game EarthBound Zero
Start Address 01E10
End Address 0200F
# of Entries 64
Entry Length 8 (0x8)
Total Length 512 bytes (0x200)
Back to the ROM map

This has data for each PSI ability in the game, and some are unused and point to the same name, but are never learned by Ninten or Ana.

Format

  • Bytes 0-1: Pointer to name
  • Byte 2: Ninten's level
  • Byte 3: Ana's level
    If the levels are 00 or FF, then that character can't learn it. Add one to the value you see here, as level 0 means level 1.
  • Byte 4: Overworld action ID
  • Byte 5: Battle action ID
  • Byte 6: Unused
  • Byte 7: PP cost

Known overworld action IDs

Unlike the battle actions, the overworld actions' using text is separate from the effect. That means the Overworld action ID only changes the effect of the item!
